# CodeAlpha Python Internship - Task 1

## Hangman Game
This is a python-based Hangman game built as part of Task 1 for the CodeAlpha Python Programming Intrnship.

### Features
- Random word selection
- Limited number of attempts
- Console-based interaction

### How to run
python hangman.py

### About
A simple console game where the player guesses letters to reveal a hidden word before running out of attempts.
